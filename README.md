# MAT401 Computational Project
MAT401 coursework project. Simulation of the rigid-body dynamics of a rotating cone via the Runge-Kutta method.

## Building and Running

This project manages its python version and dependencies with [uv](https://github.com/astral-sh/uv). It runs on Python 3.14.
